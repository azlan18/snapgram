# snapgram

## Setup Steps

## Step 1 Clone the repo
```
git clone 
```
## Step 2 Create a .env file in the root folder and add the following 
```
PORT = 5000 
MONGO_DB_URI = <your_mongo_uri>
JWT_SECRET = <your_jwt_secrte>
NODE_ENV = development
```
## Step 3 Run
```
npm run build
npm run start
```
# Step 4 Visit App on localhost:5000
